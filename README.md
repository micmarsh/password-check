# password-check

password checker library for clojure

## Usage

#### checker functions

 * not-blank?
 * contains-uppercase?
 * contains-lowercase?
 * contains-alphabet?
 * contains-symbol?
 * not-same-characters?
 * not-sequential-password?
 * not-contains-sequence?
 * not-cointains-repeats?
 * not-contains-multi-byte-character?
 * length-range
